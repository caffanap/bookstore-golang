# Welcome to BookStore!

Percobaan pertama menggunakan golang dengan framework Fiber & Gorm.
Aplikasi CRUD sederhana.


# Database

Buat database dengan nama **bookstore**

## MAKEFILE

 - BUILD : **make build**
 - RUN : **make run**
 - DEV : **make dev**

## ENV

Copy **.env.example** menjadi **.env** dan sesuaikan valuenya
